# CLIX

A utility-first, composable CLI framework.

## Installation

Add `:clix` to the list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:clix, <requirement>}
  ]
end
```

## Usage

For more information, see the [documentation](https://hexdocs.pm/clix).
